The instructions provided here are for Microsoft Windows platforms (Win7 or above).
You have to install those kind of things on your computer in order to be able to work it
1)Apache HTTP Server
2)MySQL Database Server
3)PHP,CodeIgniter
4)CodeIgniter
5)Git
6)XAMPP
After downloading that file ,first you have to change the folder name as 'ci'.
Go to Windows(C:) and find 'xampp' folder.
When you click the xampp folder ,You will see 'htdocs' folder .
Drag and drop 'ci' folder into the htdocs folder.
That is all .Before working that website ,dont forget to open XAMPP .
Otherwise you can not see the page which is searched.
